# n8n-automatic-paper-reading-agent
This is for n8n system json file.
Download it and import the json file for your n8n account.
And you can get the free agent for your paper summary every day !
( Remember to change your API key and google mail address in agent system )

This is the github link : 
https://github.com/c0970928098-tech/n8n-automatic-paper-reading-agent

This is the demo video : 
https://youtu.be/cvarMf61UeY?si=17QGJaEBgnxjq9th

How to change the topic for what you desired ?
1. Enter the node of "Message a model1"
2. Find the prompt "Give me the arxiv api of latest 7 papers about LLMs in cs.AI."
3. Change your request for the prompt.

If the agent can`t work successfully, you can do some quickly check below :
1. Check if you put your google API key or not.
2. Check if you insert your google mail address or not.
3. If you push too many papers or prompts, the agent will be crash easily.
4. The system prompt and user prompt should be similar and make sure your prompt is concise without conflict.
